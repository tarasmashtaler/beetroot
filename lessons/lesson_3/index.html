lesson№3(GIT)

На цьому уроці ми познайомились з GIT. Git – система контролю версій. Система контролю версій вона дозволяє зберігати декілька створених версій файлу, проекту чи його складових, а також бачити зміни, які були створені розробником при роботі.
Я зареєструвався на GITHUB. Мною був створений публічний репозиторій "beetroot". Він був створений для проєкту "Домашні робіти для Beetroot". За допомогою стандартної термінальної команди(git clone https://github.com/tarasmashtaler/beetroot.git .) проєкт був клонований на локальну машину. Була створена вітка проєкту "basic_html", в якій проходила частина змін. Я пройшов процес переналаштування попередніх уроків під даний проект, під час цього процесу мною додавалися(git add .), коммітилися(git commit -m "added email"), відправлялися на GITHUB зміни.

За цим посиланням ти перейдеш на мій GITHUB tarasmashtaler
